# TechShack Operations System (TCEMS)

## Project Overview
TCEMS is a premium hybrid application (Android Native + Web PWA) designed for TechShack's internal operations.

### Key Features:
- **Dashboard:** Real-time business overview.
- **Intake:** Fast device registration with offline support.
- **Repair Tickets:** End-to-end status tracking for technicians.
- **POS & Inventory:** Retail operations and parts management.
- **Payroll & HR:** Employee management and automatic payout calculation.
- **Investor Portal:** Dividend and profit reporting.

### Tech Stack:
- **Frontend:** React (Vite)
- **Framework:** Capacitor (for Android Hybrid features)
- **Database:** Supabase (PostgreSQL)
- **Aesthetic:** Linus Tech Tips (LTT) Inspired

### Hardware Integration:
- **Label Printer:** P2 (40x46mm)
- **Receipt Printer:** PT-210 (58mm)
